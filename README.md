# Data Science Learning Portfolio

This repository documents my structured journey through Data Science training, organized into progressive learning phases at Moringa School.

## Structure

- Phase 1: Python, Git, SQL, and Data Analysis Foundations  
- Phase 2: Statistics, Probability, and Inferential Analysis  
- Phase 3: Machine Learning Fundamentals and Algorithms  
- Phase 4: Advanced Machine Learning and Model Optimization  
- Phase 5: Capstone Projects and Applied Data Science  

## Purpose

This portfolio demonstrates:
- Hands-on data science projects  
- Version control using Git and GitHub  
- Progressive skill development from beginner to advanced machine learning concepts  
- Practical application of real-world datasets  

## Tools & Technologies

Python, Pandas, NumPy, Scikit-learn, SQL, Git, Jupyter Notebooks
